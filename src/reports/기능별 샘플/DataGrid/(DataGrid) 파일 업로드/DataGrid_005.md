## 1. 개요

DataGrid에 xlsx, xls, csv, txt 파일을 업로드 합니다.

## 2. 화면 구성

| 컨트롤 구분 | 컨트롤명 | 기능 |
| --- | --- | --- |
| Button | BTN_UPL | DataGrid에 파일을 업로드 |
| DataGrid | DG_IMPORT | 데이터 조회 |

## 3. 주요 API

| API | 설명 |
| --- | --- |
| `ServiceCall` | 서버 측 서비스 호출 |
| `UploadLocalFile` | 사용자의 로컬 파일을 서버로 업로드 |

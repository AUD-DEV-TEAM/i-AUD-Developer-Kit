## 1. 개요

---

_*AUTO*\_SIZE_ 예약어 사용해 MXGrid 크기에 따라 Column 및 Row의 사이즈를 가변적으로 조정

## 2. 화면 구성

---

| 컨트롤 구분 | 컨트롤명  | 기능                                      |
| ----------- | --------- | ----------------------------------------- |
| Group       | HEADER    | 화면 상단에 보고서 제목 및 조회 조건 배치 |
| Label       | LBL_TTL_1 | 보고서 제목                               |
| MXGrid      | MX        | MXGrid (엑셀) 기반 데이터 조회            |

## 1. 개요

---

_C_ 예약어 사용해 지정한 컨트롤을 해당 영역에 배치

## 2. 화면 구성

---

| 컨트롤 구분 | 컨트롤명  | 기능                                      |
| ----------- | --------- | ----------------------------------------- |
| Group       | HEADER    | 화면 상단에 보고서 제목 및 조회 조건 배치 |
| Label       | LBL_TTL_1 | 보고서 제목                               |
| Button      | BTN_REF   | 조회 수행                                 |
| MXGrid      | MX        | MXGrid (엑셀) 기반 데이터 조회            |
| DataGrid    | GRD       | DataGrid(표) 기반 데이터 시각화           |
| Chart       | CHT       | 막대 차트 기반 데이터 시각화              |

## 1. 개요

부모 보고서에서 호출되어 전달 받은 파라미터를 조회합니다.

## 2. 화면 구성

| 컨트롤 구분 | 컨트롤명 | 기능 |
| --- | --- | --- |
| RichTextBox | TXT_PARMETER | 전달 받은 파라미터를 조회 |

## 3. 주요 

| API | 설명 |
| --- | --- |
| `GetDialogRequestParams` | ShowReportDialog에서 호출된 팝업에서 부모 보고서가 전달한 파라미터 반환 |
| `GetGlobalParams` | 전역 쿼리 파라미터를 반환 |

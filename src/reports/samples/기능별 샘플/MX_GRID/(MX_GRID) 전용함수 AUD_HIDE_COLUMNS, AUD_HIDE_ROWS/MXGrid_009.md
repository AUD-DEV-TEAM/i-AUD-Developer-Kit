## 1. 개요

AUD_HIDE_COLUMNS, AUD_HIDE_ROW 함수를 이용하여 MX-Grid 내의 특정 행/열을 숨김 처리합니다.

## 2. 화면 구성

| 컨트롤 구분 | 컨트롤명 | 기능 |
| --- | --- | --- |
| CheckBox | VS_HIDE | 행/열 숨김 여부 제어 |
| MX-Grid | MX_HIDE | 데이터 조회 |

## 3. 조회 조건 및 파라미터

- 파라미터

```
VS_HIDE
```

## 4. 주요 기능

| 기능 | 설명 |
| --- | --- |
| `AUD_HIDE_COLUMNS` | 지정한 Column영역을 숨김 처리 |
| `AUD_HIDE_ROW` | 지정한 Row 영역을 숨김 처리 |

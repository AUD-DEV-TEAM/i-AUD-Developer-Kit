## 1. 개요

Button을 통해 MX-Grid의 시트 이동을 합니다.

## 2. 화면 구성

| 컨트롤 구분 | 컨트롤명 | 기능 |
| --- | --- | --- |
| Button | BTN_V1 | MX-Grid의 V1 시트로 이동 |
| Button | BTN_V2 | MX-Grid의 V2 시트로 이동 |
| Button | BTN_V3 | MX-Grid의 V3 시트로 이동 |
| Button | BTN_V4 | MX-Grid의 V4 시트로 이동 |
| MX-Grid | MX_CHANGE | 데이터 조회 |

## 3. 주요 API

| API | 설명 |
| --- | --- |
| `ChangeSheet` | MX-Grid의 특정 시트로 이동 |
